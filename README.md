Hcode 是一款基于 Tkinter 开发的轻量级 Python 代码编辑器，以 VSCode 经典深色主题为视觉基底，聚焦「简洁实用」核心，专为 Python 初学者与轻量开发场景打造，无需复杂配置即可开箱即用。

编辑器覆盖完整的 Python 开发基础需求：文件层面支持新建、打开、保存 / 另存为操作，搭配 Ctrl+N/O/S 等快捷键提升效率；编辑层面内置实时行号、智能自动缩进、多维度语法高亮（关键字 / 字符串 / 注释 / 数字分色显示）
，并支持撤销、重做、全选等基础编辑功能，贴合日常编码习惯。

界面设计兼顾视觉体验与操作便捷性，分区清晰的深色主题降低视觉疲劳，底部状态栏实时提示文件状态与操作快捷键，窗口关闭前自动检测未保存内容，避免代码丢失。整体轻量化无冗余，打包后可生成独立可执行文件，无需 Python
环境即可运行，是 Python 入门学习、小型脚本编写的优质工具。
